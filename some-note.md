register


RDI  64-bit
EDI  32-bit
DI   16-bit
DIL   8-bit

E = Extended (32-bit)
R = 64-bit register version


AX	Accumulator	Arithmetic operations, function return values
BX	Base	Base address for memory access
CX	Counter	Loop counters, shift counts
DX	Data	Extended arithmetic, I/O ports
SI	Source Index	Source pointer for string operations
DI	Destination Index	Destination pointer for string operations
